
# Metabase 

1. Downlaod java and set environment for java
2. Download metabase.jar file from metabase official website.

```
java -jar metabase.jar
```


# Change port

## Windows
SET MB_JETTY_PORT=9000


## linux
export MB_JETTY_PORT=9000
